# txfetch

A minimal system information fetcher for Termux. Bold, high-contrast and barebones.

> Note: This is build to work for Android only.

---

## Preview


## Setup 

1. **Clone the repository**
```
git clone https://github.com/RishOnBash/txfetch.git
cd txfetch
```

2. **Install**
```
make install
```

3. **Run**
```
~ $ txfetch
```
4. **Uninstall**
```
make uninstall
```
